# Shanti - Mental Wellness & Mindfulness Companion

## Name of the domain:
**Health & Personal Growth - Mental Wellness**

## The Problem:
In today's fast-paced world, mental health challenges are increasingly common, yet many people lack easy access to:
- Immediate emotional support during difficult moments
- Tools to track and understand their emotional patterns
- Guided mindfulness and meditation practices
- A safe, judgment-free space for self-reflection
- Affordable mental wellness resources available 24/7

Traditional therapy is often expensive, has long wait times, and may not be accessible when someone needs support immediately. There's a critical gap between needing help and getting it.

## Target Users:
- **Students** dealing with academic stress and anxiety
- **Working professionals** managing work-life balance and burnout
- **Individuals** seeking daily mindfulness and emotional wellness support
- **People** who want to develop better mental health habits
- **Anyone** looking for accessible, immediate emotional support

## Getting Started

### Prerequisites
- Node.js 18+ installed
- Firebase project for authentication

### Installation

```bash
# Clone the repository
git clone <your-repo-url>

# Install dependencies
npm install

# Set up environment variables
# Create a .env file with your Firebase config

# Run development server
npm run dev
```

### Firebase Setup
1. Create a Firebase project at https://console.firebase.google.com/
2. Enable Google Authentication
3. Add your Firebase config to the app
4. Update authorized domains

## Usage

1. **Sign In**: Click "Get Started" and sign in with your Google account
2. **Chat**: Talk to the AI wellness companion about how you're feeling
3. **Track Mood**: Log your daily mood to see patterns over time
4. **Meditate**: Use the meditation timer for guided sessions
5. **Customize**: Update your profile and switch languages as needed
